## Requirements

- python 3.9
- pipenv (`pip install pipenv`)


## Installation

This app is built using `pipenv`.

```
pipenv install
```


## Execution

- Windows
  ```
  pipenv run python main.py
  ```
  
- Linux
  ```
  pipenv run python3 main.py
  ```
